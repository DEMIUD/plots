# Plots

A collection of data visualizations.

## Visualizations

### [AI Compute Timeline](ai-compute-timeline/)

A publication-quality semi-log timeline showing training FLOPs for key AI milestones from 1900 to 2026.

![AI Compute Timeline](ai-compute-timeline/output/ai_compute_timeline_highres.png)

**Features:**
- Interactive HTML with hover tooltips
- Semi-log scale spanning 25+ orders of magnitude
- Color-coded by category, sized by impact
- Era shading and Moore's Law reference line

[View interactive version](ai-compute-timeline/output/ai_compute_timeline_interactive.html) | [Source code](ai-compute-timeline/src/)
